### Finext
Finext es el proyecto de fnal de curso que desarrollé junto a dos compañeros más, es una web app orientada a pymes y autónomos, de gestión de transacciones, facturas, etc...
En este proyecto use diferentes tecnologías como lo son

* React + vite (TypeScript) 
* Laravel
* MariaDB
* Anime.js
* i18next
* Apexcharts.js

Ver proyecto: https://finext.cat
--

### Fluently
Fluently es un mini-framework desde 0 hecho en PHP.
Este nació de un ejercicio de clase donde se nos dió un sistema de carpetas para poder entender el MVC, en el desarrollo del ejercicio me di cuenta de muchas cosas que faltaban por lo que me decidí en mejorarlo y hacer un mini framework.<br/>
Empecé solamente con una carpeta de controlador, modelo y vistas y lo he seguido escalando hasta el mini-framework que sigo desarrollando.

Ver proyecto: https://github.com/injerr/fluently
